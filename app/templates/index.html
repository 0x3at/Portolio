Mappa

Mappa is an open source command-line interface (CLI) tool designed to simplify data manipulation for CSV and XLSX files. It enables users to efficiently perform various operations, such as renaming header keys, remapping header positions, stripping unnecessary columns, and converting headers to lowercase. Mappa streamlines data preparation, making it an essential tool for data analysts, developers, and anyone working with spreadsheet data. It can be installed VIA pip @ mappa-cli

See Code
UNDER DEVELOPMENT

ArbEths is a primitive arbitrage interface that allows developers and traders to run multiple arbitrage bots via the command line. This application facilitates stablecoin arbitrage across various EVM-based networks and their DEXs. As an object-oriented package, ArbEths enables users to configure trading bots through dependency injection. While preset traders provide quick setup, developers can create personalized traders as customizable building blocks, enhancing flexibility and scalability in trading strategies.
